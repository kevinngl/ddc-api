## Del Donation Care API (WIP)

# Pre Requisit
- nodeJs
- npm
- mysql

# Installing
- clone repository
- run `npm install`
- run `cp .env.example .env`
- run sql query

# How To Run
- adjust your env file
- run `npm run start`
